【数论】gcd && lcm

Veröffentlicht am 2016-05-02

gcd（阿基米德）求最大公约数

用一般的方法去求最大公约数会费时很多，如果位数上升到18位数的时候，再一个个的去求，那就太浪费生命了。
那就用一个高效的gcd来解吧，代码很简单，如下：
typedef long long ll;
ll gcd(ll a,ll b)
{
if(b == 0)
return a;
return gcd(b,a%b);
}

这代码通俗易懂，很简短，却也很高能，把它牢记于心~

gcd 和 lcm 联合起来求 最小公倍数

其实这个就是用a乘b的值在除以gcd就行了，代码：

typedef long long ll;
ll gcd(ll a,ll b)
{
if(b == 0)
return a;
return gcd(b,a%b);
}
ll lcm(ll a,ll b)
{
return a*b/gcd(a,b);
}

when ten years goes.

Veröffentlicht am 2016-04-30

朋友，像一杯酒，像一首老歌 。

恍恍惚惚，高中毕业快一年了。不知道当初一起同窗共苦的你们现在过得怎么样？

深夜，总是容易让人迷醉，特别是在这样一个满是霓虹的大都市里。

##那些迷茫而深沉的日子
不知道是不是我帮你们提高了你们的酒量，呵呵~ 那时盛夏的夜晚，在6楼的楼梯里，我们就光着身子，用那几罐啤酒，聊尽了一生的往事。
是的，相信那时我们都很迷茫，未来，怎么来？
第二天起床，也许我们都感到有种罪恶感。但是现在回想起来，我感到了一种深沉。人生知知己己，哪才能找来你们两个，同我一起，在青春中染色！

##蒹葭苍苍 白露为霜
那时我们都是纯情的少年，有心上人，却都渐行渐远（当然高中毕业后超宇终成眷属），而我现在回想的不是那时的那位女孩，而是你们陪我吐露心伤的日子。当然我一直都是洒洒脱脱的样子~

##十年之约-
那天我说我们似乎少了点什么东西，就像三条各自奔向大海的三只小船，没有什么牵连，这让我感到不安。 所以我才有了十年之约的这个想法，就这样，我说了算，十年之约就生效了（至于内容—~）。

为了我们的年头，为了我们的永远的情谊，为了十年之约，来啊~ 好兄弟，干上一杯 ！
