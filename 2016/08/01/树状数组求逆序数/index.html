HDU4911 树状数组求逆序数

发表于 2016-08-01

题目

Inversion

（本题解默认你已经学过了树状数组）
Description
bobo has a sequence a 1,a 2,…,a n. He is allowed to swap two adjacent numbers for no more than k times.

Find the minimum number of inversions after his swaps.

Note: The number of inversions is the number of pair (i,j) where 1≤ia j.
Input
The input consists of several tests. For each tests:

The first line contains 2 integers n,k (1≤n≤10 5,0≤k≤10 9). The second line contains n integers a 1,a 2,…,a n (0≤a i≤10 9).
Output
For each tests:

A single integer denotes the minimum number of inversions.
Sample Input
3 1
2 2 1
3 0
2 2 1
Sample Output
1
2

解题

题意

输入n，k。然后输入n个数，求解这n个数的逆序数。
逆序数：满足：ai > aj && i < j 的两个数叫做一对逆序数对。求一组数列中所有满足逆序数对的个数就是该数列的逆序数对。
比如：5，3，4，2，1 则5和3，5和4，5和2，5和1都是一对逆序数对，改数列中共有9对逆序数对，即逆序数为9.

思路

因为该题的n_max = 100000，用树状数组求解（比较快）。
1、先把这n个数全部离散化为1到n。
2、把离散化后的1到n用树状数组进行一个个的更新，求和。
3、求出的和即为所有的逆序数，再减去k，即可得出题目要求。

对以上三个步骤进行详解

1步骤的意思是把一组数列规律化，只取它的相对大小。什么意思呢？ 举个例子就明白了。比如输入： 6， 8， 4， 7 。因为数的值的大小对我们题目没有影响，我们题目要求的只是逆序数，逆序数只用比相对大小就可以了。所以6，8，4，7 我们可以把它换成数字 2，4，1，3 （即相对大小，6是第2小的，8是最大的…）。这样说了之后是不是感觉很明白了？但是还有一个地方需要注意，如果数列中含有相同的数呢，那该怎么比较相对大小？比如说 2，2，3，1。 这时，我们应该把它离散化为2，3，4，1。看明白了吗？ 意思就是把第一个2认为是小的，第二个2认为是大的。

2步骤。采用上述的数列2，3，4，1。当输入2时，把c[2]（c是树状数组中的数列，a输入后的离散化后的数列）更新为1，则c数列变成0，0，0，1，0. 然后求和ans -= i-sum(a[2]).此时i=1，因为c中和为1，1-1=0，即还没有存在逆序数。遍历3，c变成0，0，1，1，0，sum(a[3])=2,i=2, 2-2=0 ,也还没存在逆序数。当遍历4时，同上。当遍历1时，i-sum(a[4])=3,即4-1=3；则有3个逆序数。

3步骤。因为题中的k的意思是对两个数调换k次位置。所以每调换1次，就少了1次逆序数.
就直接用步骤2求得的3-k就行了。当然如果3-k<0，则答案为0，不应该存在负数。

代码

#include <cstdio>
#include <iostream>
#include <cmath>
#include <string>
#include <cstring>
#include <algorithm>

#define ll long long
using namespace std;

int c[500005];
struct node//w用来记录输入的顺序
{
        int v, w;
};
int lowbit(int x)
{
        return x & (-x);
}
bool cmp(node x, node y)
{
        return x.v < y.v;
}

int  sum(int x)
{
        int ans = 0;
        while (x > 0)
        {
                ans += c[x];
                x -= lowbit(x);
        }
        return ans;
}

void update(int i, int v, int n)
{
        while (i <= n)
        {
                c[i] += v;
                i += lowbit(i);
        }
}
int main()
{
        node tree[100005];
        int n;
        ll k;
        while (~scanf("%d%I64d", &n, &k))
        {
                memset(c, 0, sizeof(c));
                for (int i = 1; i <= n; ++i)
                {
                        scanf("%d", &tree[i].v);
                        tree[i].w = i;//记录顺序
                }
                stable_sort(tree + 1, tree + n + 1, cmp);//把题解中说的2,2,3,1排序为1，2，2，3,即两个2的相对位置不变
                int s[100005];
                for (int i = 1; i <= n; ++i)
                {
                        s[tree[i].w] = i;//这里就是把以上排好序的1，2，2，3变成1，2，3，4
                }
                ll ans = 0;

                for (int i = 1; i <= n; i++)
                {
                        update(s[i], 1, n);//更新
                        ans += i - sum(s[i]);//求和
                }
                if (k < ans)
                {
                        printf("%I64d\n", ans - k);
                }
                else
                {
                        printf("0\n");
                }
        }

        return 0;
}
