扩展欧几里得算法

已经知道gcd可以求两个数的最大公约数。
已知一定存在 xa + xb = gcd(a,b)（x,y皆为整数） , 求解未知数x,y.

模板

int extend_gcd(int a,int b,int &x,int &y)
{
if(b == 0)//当b = 0;则此时根据xa+yb = gcd(a,b) 则gcd(a,b) = a,x = 1, y = 0;
{
x = 1;
y = 0;
return a;//返回最大公倍数
}
int ans = extend_gcd(b,a%b,x,y);//递归直到b = 0.
int t = x;
x = y;
y = t - a/b*y;
return ans;
}

原理

当实现一次递归时，a’ = b, b’ = a%b; 且有xa’ + yb’ = gcd(a’,b’);
因为 b’ = a%b = a - a/bb; gcd(a’,b’) = gcd(a,b);
所以 xb + y(a-a/bb) = gcd(a,b);
整理得 ya + b(x-a/by) = gcd(a,b);
所以 x = y, y = a-a/by; 就可以求出x,y.

题目

hdu1576

代码：

#include <cstdio>
#include <iostream>
#include <algorithm>
using namespace std;
#define BGT 9999999
#define ll long long
#define M 9973
void extend_gcd(ll a,ll b, ll &x, ll &y)
{
    if(b == 0)
    {
        x = 1;
        y = 0;
        return ;
    }
    else{
        extend_gcd(b,a%b,x,y);
        ll t = x;
        x = y;
        y = t - a/b*y;
    }
}
int main(int argc, char const *argv[])
{
    int T;
    ll n,b,x,y;
    cin>>T;
    while(T--)
    {
        cin>>n>>b;
        extend_gcd(b,M,x,y);
        ll ans = x*n;
        ans = (ans%M + M)%M;
        cout<<ans<<endl;
    }
    return 0;
}

解析：设ans=(A/B)%9973,则 ans+9973x=A/B,两边乘以B，得ansB+9973xB=A (1),
又因为 A%9973=n, 则 9973y+n=A (2)，代入（1）并整理得：
ansB/n + 9973xB/n = 1 此式子与gcd(B,9973) = 1,可有
BX + 9973Y = gcd(B,9973) = 1 （3）
则用扩展欧几里德即可求出X，则ans = X*n;
注意：此时的ans可能为负值，使他变成正值且又满足题意，则：
ans = (ans%9973+9973)%9973 （4）
因为X和Y是不定解，所以ans也是不定解。通过（4）的变换后
可以把负的变成正的，且同时满足（3）。
这里补充一点，负数的取余和mod是不一样的。
-3%2=-1,而-3mod2=1,可以理解为-3mod2=(-3%2)+2 , 这就是为什么要做（4）的变换的原因。

int init_prim(int n)  {  
    memset(visit, true, sizeof(visit));  
   int num = 0;  
   for (int i = 2; i <= n; ++i)  {  
       if (visit[i] == true)  {  
            num++;  
            prime[num] = i;  
        }  
       for (int j = 1; ((j <= num) && (i * prime[j] <= n));  ++j)  {  
            visit[i * prime[j]] = false;  
           if (i % prime[j] == 0) break; //避免重复，进一步提高效率

解析原理

把2到n内的所有素数都依次放进prime数组中。
把2的倍数4筛掉，再把3的倍数6筛掉，把5的倍数筛掉，把7的倍数筛掉 …..
筛掉的方法是visit布尔数组，如果是以上这些素数的倍数的，则让visit为false,当为相应false时，prime不存进相应的值，即筛掉！！！
