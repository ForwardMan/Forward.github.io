【数论】gcd && lcm

Posted on 2016-05-02

gcd（阿基米德）求最大公约数

用一般的方法去求最大公约数会费时很多，如果位数上升到18位数的时候，再一个个的去求，那就太浪费生命了。
那就用一个高效的gcd来解吧，代码很简单，如下：
typedef long long ll;
ll gcd(ll a,ll b)
{
if(b == 0)
return a;
return gcd(b,a%b);
}

这代码通俗易懂，很简短，却也很高能，把它牢记于心~

gcd 和 lcm 联合起来求 最小公倍数

其实这个就是用a乘b的值在除以gcd就行了，代码：

typedef long long ll;
ll gcd(ll a,ll b)
{
if(b == 0)
return a;
return gcd(b,a%b);
}
ll lcm(ll a,ll b)
{
return a*b/gcd(a,b);
}
